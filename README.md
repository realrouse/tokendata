Requires:
- Apache Server
- MySQL Database
- node.js 16


Step 1:
- Set up MySQL database
- (Optional) Set up Apache server and setup a reverse proxy to assign a domain name to the API.
- Enter details into config.js

Step 2: 
- Run `node server.js`

Step 3:
Access the API at [IPADDRESS OR LOCALHOST]:3000/api/burns
